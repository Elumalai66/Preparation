Pattern 1: +1 Letter
Example: CAT → DBU

Example: CAT → 24 → C = 3, A = 1, T = 20 → 3 + 1 + 20=24

Pattern 3: Alphabet position + combine
Example: CAT → 3120 → C = 3, A = 1, T = 20 → 3 1 20 → 3120
